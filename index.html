Artfoundry

Artfoundry is a place where creativity is kindeled, stoked, forged, and polished in order to produce an exceptional piece of form and function. Our primary focus for this creativity is mobile, web, and desktop games.

Artfoundry was originally formed back in 1997 as a Mac and PC consulting business called Creative Computer Solutions by two friends and colleagues, David Markowitz and Caven Keith.

It has since grown in scope to focus on entertainment media, starting with web and multimedia design, then moving to 3D animation, and finally taking on game design.

Now simply a portfolio site showcasing David's work, Artfoundry focuses on game development, fine art, and web application development, and the merging of these disciplines.

Betrayal is a single player module for Neverwinter Nights 2 full of deception and plot twists. It focuses on story, adventure, and gameplay strategy with mild role playing. Conversational choices that affect the outcome of play and multiple possible endings allow for a nice level of replayability.

  • Conversational decisions that can affect the outcome of the story
  • Multiple endings with numerous paths to each one
  • Puzzles to solve in order to progress
  • Challenging combat

A game of strategy that anyone can learn and play nearly anywhere.

Any was originally designed to be a simple table-top style turn-based strategy game that anyone could play anywhere (thus the name of the game). Players could use pennies, m+m's, dice, even pencil and paper. The object of the game is to reduce the number of your opponent's base units to 0. If you like Stratego and/or Magic The Gathering, you may like this game as well.

This computer version of Any is a simple single-player ASCII graphics version for the Terminal/Command line.

Any is currently freeware. You may download it and distribute it free of charge. We ask that you distribute the original zip file that contains both application and instructions. The contents of the zip file are copyright 2008 by Artfoundry and may not be modified in any way.

Web Apps

Step Up The Music is a one page app real-time, multi-user, collaborative music sequencer. Up to four people can join in to create a short looping song together! Six different instruments are available to play notes spanning two octaves and four measures (about eight seconds). Songs can be created for public or private availability and are automatically saved any time a change is made.

Technologies used:
  • HTML 5
  • CSS 3
  • JavaScript
  • jQuery
  • Firebase
Weeby Chattin'

Weeby Chattin' is a simple one page app chat service. Users can see a list of rooms, join or create a room, see a list of users in the room they've joined, and chat to their hearts' content. System messages declare when users join and leave rooms.

Technologies used:
  • HTML 5
